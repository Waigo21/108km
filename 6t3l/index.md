# 前言

最近在看六韬兵法。这是一部难得的囊括从战略到战术层面种种策略的兵书。

本文用md格式的文档编写，软件使用的是Mac下面的Mou。用GitHub托管。写整部书就跟写代码一样，所以将会是只言片语碎片化更新。力求2014年上半年写完。
